# **App Name**: RetailFlow

## Core Features:

- Role Selection: Choose a role (Cashier, Manager, Owner) to simulate access levels.
- POS Operations: Create sales, process refunds, and handle split payments (cash, card).
- Inventory Management: View stock levels, create/edit items (role-based restrictions).
- Procurement: Manage purchase orders and goods receipt notes (Manager, Owner).
- Reporting: View profit and tax reports with role-based access to cost price information.
- Stock Adjustment: Create and view stock adjustment records. Updates the master stock state, managed through redux
- AI-Powered Recommendation Tool for Stock Adjustment: Uses historical data to suggest reasons for stock adjustments.

## Style Guidelines:

- Primary color: Deep blue (#293B5F) for a professional and trustworthy feel.
- Background color: Light gray (#F0F4F8) for a clean and modern backdrop.
- Accent color: Teal (#3D948B) to highlight key actions and elements.
- Body and headline font: 'Inter' sans-serif font to maintain clean and neutral appearance, which can be universally used.
- Use minimalist, line-based icons for clarity and ease of understanding.
- Responsive layout with clear content hierarchy and intuitive navigation.
- Subtle transitions and animations to enhance user experience and provide visual feedback.
